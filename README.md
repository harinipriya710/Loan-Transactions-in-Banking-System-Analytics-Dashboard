# Loan-Transactions-in-Banking-System-Analytics-Dashboard

An interactive Power BI dashboard developed to analyze loan transaction data, monitor loan performance, and generate actionable business insights for banking operations.

## Features
- Loan Transaction Analysis
- Loan Approval and Repayment Monitoring
- Customer Trend Analysis
- Interactive Filters and Slicers
- KPI Tracking and Performance Metrics
- Dynamic Visualizations and Drill-Down Reports

## Technologies Used
- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization

## Key Insights
- Loan approval and repayment trends
- Customer transaction behavior analysis
- Banking performance monitoring
- KPI-based decision support
